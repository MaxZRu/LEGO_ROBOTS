Расшифровка показателей таблицы

  • Игры - количество сыгранных матчей.
  • Поб. - количество побед.
  • Нич. - количество ничьих.
  • Пор. - количество поражений.
  • Заб. - забитые мячи.
  • Проп. - пропущенные мячи.
  • Разн. - разница мячей (забитые минус пропущенные).
  • Очки - турнирные очки команды.
  • Место - позиция команды в итоговой таблице.

Критерии определения мест

Базовая система очков Победа - 3, ничья - 1, поражение - 0 (если организаторы не утверждают иное)
